# 项目运行说明

## 方式一：使用 HBuilderX（推荐）

HBuilderX 是 uniapp 官方推荐的开发工具，使用最简单。

### 步骤：

1. **下载安装 HBuilderX**
   - 访问：https://www.dcloud.io/hbuilderx.html
   - 下载并安装标准版或 App 开发版

2. **导入项目**
   - 打开 HBuilderX
   - 文件 → 导入 → 从本地目录导入
   - 选择项目目录

3. **运行项目**
   - **H5 运行**：点击顶部菜单"运行" → "运行到浏览器" → 选择 Chrome/Edge
   - **微信小程序**：点击"运行" → "运行到小程序模拟器" → "微信开发者工具"
   - **App**：点击"运行" → "运行到手机或模拟器"

---

## 方式二：使用命令行（需要配置环境）

### 前提条件：
- 已安装 Node.js（v14 或更高版本）
- 已安装 npm 或 yarn

### 步骤：

#### 1. 安装依赖

```bash
npm install
```

或者使用 yarn：
```bash
yarn install
```

#### 2. 运行项目

**H5 开发模式：**
```bash
npm run dev:h5
```
然后在浏览器访问：http://localhost:8080

**微信小程序开发模式：**
```bash
npm run dev:mp-weixin
```
编译完成后，使用微信开发者工具打开 `unpackage/dist/dev/mp-weixin` 目录

**App 开发模式：**
```bash
npm run dev:app
```

#### 3. 构建生产版本

**构建 H5：**
```bash
npm run build:h5
```
构建产物在 `unpackage/dist/build/h5` 目录

**构建微信小程序：**
```bash
npm run build:mp-weixin
```
构建产物在 `unpackage/dist/build/mp-weixin` 目录

## 常见问题

### 1. 如果使用命令行方式，但命令不存在

标准的 uniapp 项目通常使用 `@dcloudio/vite-plugin-uni`，但可能需要额外的 CLI 工具。

**解决方案：**
```bash
# 安装全局 uniapp CLI
npm install -g @dcloudio/uni-cli

# 或者使用 npx
npx @dcloudio/vite-plugin-uni
```

### 2. 微信小程序需要微信开发者工具

- 下载地址：https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html
- 使用命令行运行微信小程序前，确保微信开发者工具已安装并登录

### 3. HBuilderX 运行小程序时提示找不到微信开发者工具

在 HBuilderX 中配置微信开发者工具路径：
- 工具 → 设置 → 插件配置 → 微信开发者工具路径

## 推荐方式

**初次开发推荐使用 HBuilderX**，因为：
- ✅ 安装简单，开箱即用
- ✅ 内置调试工具
- ✅ 支持一键运行到多端
- ✅ 自动处理依赖和编译

**有 Node.js 经验的开发者可以使用命令行方式**，更适合 CI/CD 和自动化部署。

---

## 快速开始

如果使用 HBuilderX：
1. 打开 HBuilderX
2. 导入项目
3. 点击"运行" → "运行到浏览器" → 选择浏览器
4. 开始开发！

如果使用命令行：
1. 在项目目录打开终端
2. 运行 `npm install`
3. 运行 `npm run dev:h5`
4. 浏览器会自动打开项目
